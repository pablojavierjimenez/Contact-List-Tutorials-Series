API crud with NodeJS, ExpressJS and MySQL
=========================================

### Prerrequisits
Install NodeJS and NPM (here we use [Node version 12.16.3](https://nodejs.org/docs/latest-v12.x/api/) )
```bash
:~$ sudo apt install -y nodejs
:~$ sudo apt install npm
```
